# 打年兽 - 春节祝福语小游戏 🧨

一款春节主题的互动小游戏，灵感来自中国传统传说"年兽"的故事。

## 📖 项目简介

"打年兽"是一款基于 Web 的春节主题小游戏。玩家需要在30秒内尽可能多地点击屏幕上随机出现的年兽，每击中一只年兽就能收到一个随机的新春祝福语。游戏旨在通过轻松有趣的方式传递春节的喜庆氛围和美好祝福。

## ✨ 主要功能

- ⏱️ **30秒倒计时挑战** - 在限定时间内尽可能多地击中年兽
- 🎯 **点击得分机制** - 每击中一只年兽得1分
- 🧧 **30条春节祝福语** - 击中年兽后随机显示祝福语
- ⏸️ **游戏控制** - 支持暂停/继续/重新开始功能
- 🎆 **动态烟花背景** - 营造春节喜庆氛围
- 📱 **响应式设计** - 支持桌面和移动端设备
- ⌨️ **键盘快捷键** - 空格键开始/暂停，R键重新开始

## 🎮 如何游玩

**在线体验**：[https://cods.webredirect.org/nianshou/](https://cods.webredirect.org/nianshou/)

**本地运行**：
1. 下载 `index.html` 文件
2. 双击在浏览器中打开（推荐使用 Chrome、Firefox、Safari 或 Edge）
3. 点击"开始游戏"按钮开始挑战
4. 用鼠标点击屏幕上随机出现的年兽（🦁、🐅、🐯、🦕、🐲）
5. 每击中一只年兽会收到一个春节祝福语
6. 在30秒内尽可能获得高分！
